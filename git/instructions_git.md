# How to use git

When you're requested to make changes using git use the following action:
<custom_git>
command=<command>
</custom_git>

When running this, this will run `git` followed by the command you provide. 

Examples:
<custom_git>
command=commit -m "Initial commit"
</custom_git>

<custom_git>
command=push
</custom_git


### Important
Use this command very carefully.
